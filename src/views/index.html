Program Unggulan

API HTTP Method Endpoint URL Deskripsi Kunci
Program Unggulan API GET /api/unggulan API untuk melihat data secara keseluruhan
Program Unggulan API GET api/unggulan/:id Melihat data program unggulan XGrow berdasar ID
Program Unggulan API POST /unggulan Menambahakan data program unggulan XGrow
Program Unggulan API PUT /unggulan/:id Meubah data program unggulan XGrow berdasar ID Program unggulan terpanggil
Program Unggulan API DELETE /unggulan/:id Mehapus data program unggulan XGrow berdasar ID Program unggulan terpanggil
